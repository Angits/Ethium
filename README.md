### How to Install
1. U need installed python>=3.10 & git
1. `git clone https://github.com/Angits/Ethium.git`
2. Set ur bot config in ethium.py (;13 - ;19)
3. `pip install -r requirements.txt`
4. `cd src && python3 ethium.py`
